Generating realistic synthetic option prices requires implied volatility (IV) as an input, yet IV itself is derived from observed option prices---creating a circular dependency that limits synthetic data availability.
In this study, we addressed this challenge by developing a three-stage pipeline that produces self-consistent American option prices: (i)~a Jump Hidden Markov Model (JumpHMM) generated realistic multi-asset price paths with heavy tails, volatility clustering, and cross-asset dependence via a Student-$t$ copula; (ii)~a modified Heston stochastic variance process converted those price paths into IV, where the mean-reversion target $\theta$ was driven by the HMM regime state, days to expiration, moneyness, and an aggregate market mood indicator; and (iii)~a Cox--Ross--Rubinstein binomial tree priced American options with early exercise using the pre-computed IV.
A central design choice was the equilibrium initialization $v_0 = \theta(s_0, \mathrm{DTE}, K/S_0, M_0)$: each strike--expiration pair started at its own mean-reversion target, so the smile, skew, and term structure of the IV surface emerged automatically without external calibration data.
We calibrated a four-parameter smile function on NVDA options and demonstrated that the smile shape generalized across AMD, MU, and INTC with only a scalar level adjustment per ticker (cross-ticker RMSE of 1--5\% IV).
The framework is implemented as an open-source Julia package designed to serve as a synthetic data generator for downstream machine-learning and risk-analysis applications.
